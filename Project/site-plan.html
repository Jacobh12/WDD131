Overview

Purpose

The purpose of this website is to draw pictures in the website and to hopefully be able to save them to a second page on the site.

Audience

The Audience for this website would be anyone that likes to draw and wants a free drawing pad.

Dynamic elements

The dynamic elements in this webpage are the canvas and the brush needed to draw. Other elements can include buttons that draw simple shapes randomly on the canvas.
